# Баги приложения dumb.exe найденые при тестировании по mindmap
---

## Баг 1. Интерфейс зависает на 3 секунды при удалении элемента

**Как воспроизвести:** выделить элемент в списке → нажать кнопку "Delete"

**Ожидание:** элемент удаляется мгновенно

**Факт:** интерфейс полностью замирает на 3 секунды. Затем элемент удаляется.

---

## Баг 2. Сортировка работает только по первым 2 буквам

**Как воспроизвести:** добавить элементы 'aab', 'aaa', 'aav' → нажать "Sort"

**Ожидание:** элементы отсортируются полностью — 'aaa', 'aab', 'aav'(a-z) или 'aav', 'aab', 'aaa' (z-a)

**Факт:** элементы начинающиеся на 'aa' расположены в порядке ввода


---

## Баг 3. Фильтр "подстрока" скрывает элементы с повторным вхождением

**Как воспроизвести:** добавить элементы  'ap', 'ab', 'ff', 'aa' → выбрать фильтр "Apply filter to any substring" → ввести фильтр "a"

**Ожидание:**  видны  'ap', 'ab', 'aa'

**Факт:** видны  'ap', 'ab. Элемент с двойным вхождением подстроки фильтр исключает.

---
## Баг 4. Фильтр работает неверно

**Как воспроизвести:** Добавить 'ab', 'bb', 'aa' → выбрать фильтр "Apply filter to any substring" → Ввести фильтр 'b' по концу

**Ожидание:**  видны  'ab', 'bb'

**Факт:** видны 'ab'. Элемент с двойным вхождением конца фильтр исключает.

---


## Баг 5. Лимит по фильтрам работает неверно

**Как воспроизвести:** Добавить 11 фильтров для конца

**Ожидание:** ошибка при 11-м

**Факт:** заменяет 10 фильтр на 11

---

## Баг 6. "New" не очищает список и фильтры в комбобоксах

**Как воспроизвести:** добавить фильтры → нажать File → New

**Ожидание:** полный сброс — список пуст, фильтры пусты

**Факт:** список и фильтры не очищены

---

## Баг 7. При вводе элемента его нельзя отредактировать

**Как воспроизвести:** Ввести значение элемента и нажать на клавиатуре Backspace

**Ожидание:** удалиться последний символ

**Факт:** Backspace не работает
